# Termux-setup
This tool setup your Termux.

## how to install

Install python && pip

```bash
  pkg isntall python3
```
```bash
  pkg install python3-pip
```
install the tool ⇓

```bash
 git clone https://github.com/Halal-AsHacker/termux-setup
```
Now ⇓

```bash
  cd termux-setup
```
Install the all requirement ⇓

```bash
  pip install -r requiremen.txt
```
At last ,Now  you can run the tool ⇓

```bash
  python update.py
```
